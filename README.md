# Baekjoon_Auto_Notion

### ❗백준 문제를 풀면 자동 으로 노션에 페이지 생성해주는 프로그램입니다.


### 생성 페이지 구성
1. 문제 번호 - 제목
2. 티어 아이콘
3. 문제 링크
4. 문제 태그
5. 본인 코드 (언어별 Syntax Highlighting)
6. GPT-4의 코드 설명
